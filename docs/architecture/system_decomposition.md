# System Decomposition

## Modules
- Authentication
- Menu / Catalog
- Cart Management
- Order Management
- Payment Processing
- User Interaction

## Component Responsibilities
- **Authentication**: Handles user login and authentication.
- **Menu / Catalog**: Displays available food items and supports search and filtering.
- **Cart Management**: Manages adding, removing, and viewing items in the cart.
- **Order Management**: Handles checkout and order processing, including delivery estimates.
- **Payment Processing**: Manages payment methods and transaction handling.
- **User Interaction**: Supports contact, help enquiries, and user reviews.
