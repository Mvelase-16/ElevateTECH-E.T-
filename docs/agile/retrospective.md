# Sprint 1 Retrospective

During Sprint 1 the team focused on designing the basic structure of the Cafeteria Pre-Order Web Application.

Completed tasks:
- Project repository created
- System wireframes designed
- Basic homepage implemented
- Database structure designed

The team demonstrated the early interface showing how students will view the food menu and place orders.
